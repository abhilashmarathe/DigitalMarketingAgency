# DigitalMarketingAgency

A Django based web application for showcasing digital marketing services.

## Features
- Service pages (SEO, social media etc.)
- Contact/lead form
- Admin dashboard

## Tech Stack
Python, Django, HTML, CSS

## Setup
1. Clone repo  
2. `pip install -r requirements.txt`  
3. `python manage.py migrate`  
4. `python manage.py runserver`
